# Copella Top Creators Plugin

Расширенная система страниц креаторов для WordPress с интеграцией с Topics Plugin.

## Возможности

### 🎨 Единый стиль дизайна
- Размытый фон аватарки в заголовке страницы
- Современный дизайн в стиле Copella
- Адаптивная верстка для всех устройств
- Анимации и переходы

### 👤 Управление креаторами
- Custom Post Type для креаторов
- Категории креаторов
- Детальная информация о креаторе:
  - Возраст и опыт работы
  - Специализация
  - Достижения и заслуги
  - Социальные сети с иконками

### 🔗 Интеграция с Topics Plugin
- Привязка плейлистов к креаторам
- Связь с авторами Topics Plugin
- Отображение проектов (видео и стримы)
- Двусторонняя связь между плагинами

### 📱 Шорткоды
- `[creator_card]` - Карточка креатора с кнопкой копирования
- `[creators_list]` - Список креаторов
- `[creator_projects]` - Проекты креатора

### 🌐 Полноценные страницы
- **Страница креатора** - автоматически создается при публикации креатора
- **Архив креаторов** - список всех креаторов по адресу `/creators/`

## Установка

1. Загрузите плагин в папку `/wp-content/plugins/`
2. Активируйте плагин в админке WordPress
3. Перейдите в меню "Креаторы" для настройки

## Использование

### Создание креатора

1. Перейдите в **Креаторы → Добавить креатора**
2. Заполните основную информацию:
   - Название (имя креатора)
   - Аватарка
   - Описание
3. Заполните детали:
   - Возраст
   - Опыт работы
   - Специализация
   - Достижения
4. Добавьте социальные сети
5. Привяжите плейлисты из Topics Plugin
6. Опубликуйте

### Настройка социальных сетей

Формат: `Название | Ссылка | Иконка`

Пример:
```
YouTube | https://youtube.com/@username | 🎥
Telegram | https://t.me/username | 📱
Instagram | https://instagram.com/username | 📸
Twitter | https://twitter.com/username | 🐦
TikTok | https://tiktok.com/@username | 🎵
Twitch | https://twitch.tv/username | 🎮
```

### Полноценные страницы

#### Страница креатора
При создании креатора автоматически создается полноценная страница по адресу `/creators/creator-slug/`

#### Архив креаторов
Список всех креаторов доступен по адресу `/creators/`

#### Карточка креатора
```
[creator_card id="123"]
```

Параметры:
- `id` - ID креатора
- `slug` - Слаг креатора
- `show_social` - Показывать социальные сети (true/false)
- `show_achievements` - Показывать достижения (true/false)
- `show_playlists` - Показывать плейлисты (true/false)
- `show_projects` - Показывать проекты (true/false)
- `style` - Стиль (default/compact/minimal)
- `copy_shortcode` - Кнопка копирования (true/false)

#### Список креаторов
```
[creators_list category="gaming" limit="6" style="grid"]
```

Параметры:
- `category` - Категория креаторов
- `limit` - Количество креаторов
- `style` - Стиль отображения (grid/list/carousel)
- `show_social` - Показывать социальные сети (true/false)
- `show_achievements` - Показывать достижения (true/false)
- `orderby` - Сортировка (date/title/menu_order)
- `order` - Порядок (ASC/DESC)

#### Проекты креатора
```
[creator_projects id="123" type="video" limit="6"]
```

Параметры:
- `id` - ID креатора
- `slug` - Слаг креатора
- `type` - Тип проектов (all/video/stream)
- `limit` - Количество проектов
- `style` - Стиль отображения (grid/list)

## Интеграция с Topics Plugin

### Связывание авторов с креаторами

1. При редактировании автора в Topics Plugin выберите связанного креатора
2. Или при редактировании креатора выберите авторов для связи
3. Проекты авторов будут отображаться на странице креатора

### Настройки интеграции

Перейдите в **Креаторы → Интеграция** для настройки:
- Автоматическая связь авторов с креаторами
- Отображение информации о креаторе в Topics Plugin
- Категория по умолчанию для новых креаторов

## Категории креаторов

По умолчанию создаются категории:
- Гейминг
- Образование
- Развлечения
- Технологии
- Образ жизни

Вы можете создавать дополнительные категории в **Креаторы → Категории креаторов**.

## Стили

Плагин использует единый стиль Copella:
- Шрифты Gilroy
- Цветовая схема: #171717 (фон), #FF653A (акцент)
- Скругленные углы и тени
- Адаптивная верстка

## Требования

- WordPress 5.0+
- PHP 7.4+
- Topics & Playlists Plugin (для полной функциональности)
