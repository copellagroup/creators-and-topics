# Copella Topics & Playlists Plugin

Расширенная система тем, плейлистов, авторов и креаторов для WordPress. Включает Custom Post Types, социальные сети, достижения, привязку плейлистов и проектов. Единый стиль с размытым фоном аватарки.

## Возможности

### 🎯 Основные функции
- **Темы** - Custom Post Type для создания тем
- **Плейлисты** - Привязка плейлистов к темам
- **Авторы** - Система авторов с социальными сетями
- **Креаторы** - Расширенная система креаторов (НОВОЕ!)

### 🎨 Дизайн
- Единый стиль во всех компонентах
- Шрифты Gilroy
- Размытый фон аватарки
- Адаптивная верстка
- Темная тема

### 📱 Шорткоды

#### Темы и плейлисты
- `[hot_topics]` - Блок "Горячие темы"
- `[topic_playlist]` - Отображение плейлиста
- `[playlist_authors]` - Авторы плейлиста
- `[video_authors]` - Авторы видео
- `[stream_authors]` - Авторы стримов

#### Креаторы (НОВОЕ!)
- `[creator_card]` - Карточка креатора с кнопкой копирования
- `[creators_list]` - Список креаторов

### 🌐 Полноценные страницы

#### Страницы креаторов
- **Страница креатора** - автоматически создается при публикации креатора
- **Архив креаторов** - список всех креаторов по адресу `/creators/`

## Установка

1. Загрузите плагин в папку `/wp-content/plugins/`
2. Активируйте плагин в админке WordPress
3. Перейдите в меню "Креаторы" для начала работы

## Использование

### Создание креатора

1. Перейдите в **Креаторы → Добавить креатора**
2. Заполните основную информацию:
   - Название (имя креатора)
   - Описание
   - Аватарка
   - Краткое описание

3. В мета-боксах заполните:
   - **Социальные сети** - ссылки на профили
   - **Детали креатора** - возраст, опыт, специализация, достижения
   - **Привязанные плейлисты** - выберите плейлисты для отображения

### Социальные сети

Формат ввода социальных сетей:
```
YouTube | https://youtube.com/@username | 🎥
Telegram | https://t.me/username | 📱
Instagram | https://instagram.com/username | 📸
Twitter | https://twitter.com/username | 🐦
TikTok | https://tiktok.com/@username | 🎵
Twitch | https://twitch.tv/username | 🎮
```

### Полноценные страницы

#### Страница креатора
При создании креатора автоматически создается полноценная страница по адресу `/creators/creator-slug/`

#### Архив креаторов
Список всех креаторов доступен по адресу `/creators/`

#### Карточка креатора
```
[creator_card id="123"]
```

Параметры:
- `id` - ID креатора
- `slug` - Слаг креатора
- `show_social` - Показывать социальные сети (true/false)
- `show_achievements` - Показывать достижения (true/false)
- `show_playlists` - Показывать плейлисты (true/false)
- `show_projects` - Показывать проекты (true/false)
- `style` - Стиль карточки (default/compact/minimal)
- `copy_shortcode` - Показывать кнопку копирования (true/false)

#### Список креаторов
```
[creators_list category="gaming" limit="6" style="grid"]
```

Параметры:
- `category` - Категория креаторов
- `limit` - Количество креаторов
- `style` - Стиль отображения (grid/list/carousel)
- `show_social` - Показывать социальные сети (true/false)
- `show_achievements` - Показывать достижения (true/false)
- `orderby` - Сортировка (date/title/menu_order)
- `order` - Порядок (ASC/DESC)

## Интеграция

### Связь с авторами
Креаторы автоматически связываются с авторами через мета-поле `_copella_author_creator_id`

### Привязка плейлистов
В админке креатора можно выбрать плейлисты для отображения на странице

### Проекты
Автоматически отображаются проекты (видео и стримы), связанные с креатором

## Технические детали

### Custom Post Types
- `creator` - Креаторы
- `topic` - Темы
- `topic_playlist` - Плейлисты
- `topic_author` - Авторы

### Taxonomies
- `creator_category` - Категории креаторов
- `author_category` - Категории авторов

### Мета-поля креаторов
- `_copella_creator_social` - Социальные сети
- `_copella_creator_age` - Возраст
- `_copella_creator_experience` - Опыт работы
- `_copella_creator_achievements` - Достижения
- `_copella_creator_specialization` - Специализация
- `_copella_creator_playlists` - Привязанные плейлисты
